# Personal Introduction Program 

This is a simple Python program that asks the user a few questions and then displays a friendly, personalized welcome message.

## Features

- Uses `input()` to collect:
  - Name
  - Age
  - Favorite hobby
  - City
- Stores answers in variables
- Uses `print()` and f-strings to display a customized message
- Friendly and welcoming output
